## Project Overview

This is a GitHub Pages static website repository. The site is served directly from the repository at `yongun-iotai.github.io`.

## Development

No build process required. Edit HTML/CSS/JS files directly and push to deploy via GitHub Pages.

To preview locally, open `index.html` in a browser or use a local server:
```bash
python3 -m http.server 8000
```
