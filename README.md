## Overview

개인 포트폴리오 웹사이트입니다. 모던한 다크 테마와 애니메이션을 적용하여 제작되었습니다.

**Live Demo:** [https://yongun-iotai.github.io](https://yongun-iotai.github.io)

## Features

- **모던 다크 테마** - 눈에 편한 다크 모드 디자인
- **애니메이션 배경** - 동적인 그라데이션 효과
- **반응형 디자인** - 모바일, 태블릿, 데스크탑 지원
- **인터랙티브 UI** - 호버 효과 및 부드러운 스크롤

## Local Development

```bash
# Clone
git clone https://github.com/yongun-iotai/yongun-iotai.github.io.git

# Open in browser
open index.html

# Or use local server
python3 -m http.server 8000
```
